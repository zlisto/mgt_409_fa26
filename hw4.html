Homework 4: AI Workflow Agent

Covers Lectures 7–8  |  Core MBA lens: Economics

Take a real business process, map where AI belongs, build a multi-step workflow agent, then price it at scale.

Part 1 — Process map

  1. Choose a process (e.g., expense reimbursement, vendor onboarding, lead qualification, customer refund).
  2. Submit a workflow diagram (swim lanes OK): human steps, AI-assist steps, and AI-automated steps.
  3. Mark at least one step that must stay human-only and explain why.

Part 2 — Build the workflow agent

  1. Implement at least three connected steps using prior course skills (extract, RAG, research, structured output).
  2. Include structured handoffs (JSON or tables) and logs showing which step ran.
  3. Output a 1-page executive summary a manager could read.

Part 3 — Tokenomics

  1. Run the same workflow on two models or configs.
  2. Estimate cost per 100 runs and note when you would use the cheaper option.

Submission

Submit hw4.zip with code, requirements.txt, workflow diagram, sample run output, executive summary, and cost comparison (half page).
