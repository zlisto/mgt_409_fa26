Homework 5: Accounting Document Agent

Covers Lectures 9–10  |  Core MBA lens: Accounting

Ops finance: turn messy documents into books-ready fields. Optional stretch ties in messaging or MCP from class.

Required — Invoice / receipt extractor

  1. Build an agent that accepts invoice or receipt images/PDFs.
  2. Output ledger-ready JSON: vendor, date, line items, total, tax fields.
  3. Include a short note on accuracy limits and one failure case.

Optional stretch (+bonus credit)

  • Telegram or Slack bot: user sends expense photo → parsed fields returned.
  • Or: MCP workflow (e.g., Gmail + another connector) that supports an accounting or ops task.

Submission

Submit hw5.zip with code, requirements.txt, and sample extracted JSON for at least two documents.
