Homework 6: Investor Demo Site

Covers Lectures 11–12  |  Core MBA lens: Investor (stakeholder demo)

Package prior agent work into a polished Vite app with a live URL—the “Appendix: working product” slide every pitch deck wants.

Requirements

  1. Build a Vite app with at least two tabs wired to agent modules from earlier homework (e.g., doc extractor + catalog RAG, or research + decision agent).
  2. Deploy to Vercel (or equivalent) with a working public URL.
  3. README with setup instructions and env var documentation (no API keys in repo).
  4. 1 paragraph: who would use this demo and what business decision it enables.

Stretch

Same backend reachable from Telegram/Slack and the web app, or a simple mobile wrapper (PWA/Expo).

Submission

Submit GitHub repo link + live URL + paragraph via Canvas (or hw6.zip if zip submission preferred).
